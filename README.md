Exercício de Responsividade - FIAP 

Projeto prático desenvolvido para a disciplina de Front-End, com foco em CSS Media Queries e design responsivo. O objetivo foi adaptar um layout estático para dispositivos móveis e tablets.

##  Requisitos do Projeto
1. Layout adaptável para 768px com Menu Hambúrguer.
2. Imagens centrais e responsivas.
3. Tipografia adaptável conforme a resolução.
4. Header com logotipo personalizado.
5. Footer com link direto para o perfil do GitHub.
6. Criação de página "About Me" (Sobre Mim).
7. Criação de página de "Contato" com formulário funcional.

## Tecnologias Utilizadas
* **HTML5**: Estruturação semântica das páginas.
* **CSS3**: Estilização e uso de `@media queries` para responsividade.
* **Git/GitHub**: Versionamento de código.
